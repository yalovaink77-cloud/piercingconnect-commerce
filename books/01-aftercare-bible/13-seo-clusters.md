# Editorial Rules

1. Problem-urgency clusters (`infected piercing`, `piercing bump`) require cautious language and minimal affiliate CTAs.
2. Comparison clusters must use repository trust scores and ingredients—not commission rates.
3. Aftercare guides link to structured piercing type and problem records for consistency.
4. No search volume or CPC claims in published content unless verified externally outside this repository.
