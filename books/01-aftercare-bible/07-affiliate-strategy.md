# Purpose

How PiercingConnect evaluates affiliate opportunities without compromising trust.

This is a **strategy chapter** at the original Book 1 slot 07. The healing stages knowledge chapter also uses number 07: see [`07-healing-stages.md`](07-healing-stages.md).

---

# Core Principle

Affiliate revenue follows trust — not the reverse.

Evaluation order: Safety → Evidence → Professional standards → Trust → Quality → Availability → Affiliate

# Data Separation

- Product trust scores are independent of `affiliate.available`
- Commission rates are **not stored** in the repository until officially verified
- Unverified programs use `unknown` affiliate program ID
- Keyword cluster `affiliate_opportunity` is an editorial estimate — not a ranking instruction

---

# Content Rules by Intent

| Search intent | Affiliate approach |
|---------------|-------------------|
| urgent-problem-solving | Minimal or no product CTAs |
| informational | Education first; soft commercial links |
| commercial / comparison | Disclosure required; evidence-based ranking |

# Disclosure

All commercial content must include clear affiliate disclosure. PCME templates include `{{affiliate_disclosure}}`.
