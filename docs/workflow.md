---

# Definition of Done

A knowledge asset is considered complete when:

- Research completed
- Sources documented
- Schema followed
- Structured data created
- Book content updated
- Standards reviewed
- Ready for PC Media Engine

---

# Update Rule

Knowledge is never considered final.

Every entity should be reviewed whenever:

- Products change
- Scientific guidance changes
- Affiliate programs change
- Regulations change
- Better evidence becomes available

---

# Final Principle

Research once.

Structure once.

Publish everywhere.
